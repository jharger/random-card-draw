# Random Tile Bag

A program for testing board game mechanics by simulating drawing a random tile from a bag.

## Installation

This project uses Poetry for dependency management. To install:

```bash
poetry install
```

## Usage

```bash
poetry run python main.py
```

## Development

To run tests:
```bash
poetry run pytest
```

To format code:
```bash
poetry run black .
```

To lint code:
```bash
poetry run flake8
```
